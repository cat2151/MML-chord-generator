# MML-chord-generator
MMLで楽にコード進行を鳴らしてみたい。コード進行を入力したら和音MMLを出力するようなMML生成器を作ってみる

# Flashについて
演奏にFlashを使っているので、Flash非対応ブラウザの場合は音が鳴りません
